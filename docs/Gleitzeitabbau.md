# 3.2 – Gleitzeitabbau in der TimeMoto Cloud

Dieses Dokument beschreibt, wie Sie einen Antrag auf Gleitzeitabbau (Überstundenabbau) in der TimeMoto Cloud stellen.

---

## Schritt 1: Profil öffnen

Melden Sie sich in der [TimeMoto Cloud](https://cloud.timemoto.com/#/) an und klicken Sie oben rechts auf Ihren Namen, um Ihr Profil zu öffnen.

![Profil öffnen](../assets/Ueberstundenabbau/Bildschirmfoto%202026-04-29%20um%2022.27.38.png)

---

## Schritt 2: Abwesenheit beantragen

Wechseln Sie im Profil auf den Reiter **„Abwesenheiten"** und klicken Sie auf die Schaltfläche **„Abwesenheit beantragen"**.

![Abwesenheit beantragen](../assets/Ueberstundenabbau/Bildschirmfoto%202026-04-29%20um%2022.33.40.png)

---

## Schritt 3: Gleitzeitabbau ausfüllen

Füllen Sie das Formular wie folgt aus:

1. Wählen Sie als **Abwesenheitstyp** den Eintrag **„Gleitzeitabbau"**.
2. Prüfen Sie, dass im Feld **Empfänger** **Philipp Dölker** eingetragen ist.
3. Aktivieren Sie **„Ganzer Tag"**, falls der Abbau ganze Tage umfasst.
4. Tragen Sie **Startdatum** und **Enddatum** ein.
5. Geben Sie optional eine **Beschreibung** ein (z. B. „Abbau Überstunden").
6. Klicken Sie auf **„Abwesenheit beantragen"**.

![Gleitzeitabbau ausfüllen](../assets/Ueberstundenabbau/Bildschirmfoto%202026-04-29%20um%2022.36.37.png)

Der Antrag wird anschließend an Philipp Dölker zur Genehmigung weitergeleitet.
