# 2 – Zeitkorrektur in der TimeMoto Cloud

Dieses Dokument beschreibt, wie Sie eine fehlerhafte Stempelung (z. B. vergessenes Ausstempeln) in der TimeMoto Cloud korrigieren.

---

## Schritt 1: Berichte öffnen

Melden Sie sich in der [TimeMoto Cloud](https://cloud.timemoto.com/#/) an und klicken Sie oben in der Navigation auf **„Berichte"**.

![Berichte öffnen](../assets/Zeitkorrektur/Bildschirmfoto%202026-05-01%20um%2021.44.51.png)

---

## Schritt 2: Zeitbericht aufrufen und Eintrag auswählen

Wechseln Sie im Berichtsbereich auf den Reiter **„Zeitbericht"** und stellen Sie die **Tagesansicht** ein. Wählen Sie oben den gewünschten **Gehaltszeit­raum** aus.

In der Tabelle erscheinen alle Stempelungen. Klicken Sie auf die **Uhrzeit** in der Spalte „Hinein" oder „Hinaus" des zu korrigierenden Eintrags.

![Zeitbericht – Uhrzeit anklicken](../assets/Zeitkorrektur/Bildschirmfoto%202026-05-01%20um%2021.45.37.png)

---

## Schritt 3: Korrektur eintragen und Anfrage senden

Es öffnet sich das Dialogfenster **„Anwesenheit bearbeiten/speichern"**. Nehmen Sie die Korrekturen vor:

1. Tragen Sie im Feld **„Beginn"** (HINEIN) bzw. **„Ende"** (HINAUS) die korrekte Uhrzeit ein – je nachdem, welche Stempelung fehlerhaft ist.
2. Klicken Sie auf **„+ Kommentar hinzufügen"** und geben Sie eine kurze Begründung ein (z. B. *Korrektur ausstempeln*).
3. Stellen Sie sicher, dass im Feld **„Genehmiger auswählen"** **Philipp Dölker** ausgewählt ist.
4. Klicken Sie auf **„Stempelanfrage senden"**.

![Korrektur eintragen und senden](../assets/Zeitkorrektur/Bildschirmfoto%202026-05-01%20um%2021.48.49.png)

Die Korrekturanfrage wird anschließend an Philipp Dölker zur Genehmigung weitergeleitet.
