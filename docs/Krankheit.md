# 3.3 – Krankmeldung in der TimeMoto Cloud

Dieses Dokument beschreibt, wie Sie eine Krankmeldung in der TimeMoto Cloud erfassen.

> **Hinweis:** Die Erfassung in TimeMoto ersetzt nicht die persönliche Krankmeldung in der Praxis. Sie dient ausschließlich der Abbildung in der Zeitwirtschaft. Bitte informieren Sie die Praxis weiterhin wie gewohnt über Ihre Abwesenheit.

---

## Schritt 1: Profil öffnen

Melden Sie sich in der [TimeMoto Cloud](https://cloud.timemoto.com/#/) an und klicken Sie oben rechts auf Ihren Namen, um Ihr Profil zu öffnen.

![Profil öffnen](../assets/Krankheit/Bildschirmfoto%202026-04-29%20um%2022.27.38.png)

---

## Schritt 2: Abwesenheit beantragen

Wechseln Sie im Profil auf den Reiter **„Abwesenheiten"** und klicken Sie auf die Schaltfläche **„Abwesenheit beantragen"**.

![Abwesenheit beantragen](../assets/Krankheit/Bildschirmfoto%202026-04-29%20um%2022.33.40.png)

---

## Schritt 3: Krankmeldung ausfüllen

Füllen Sie das Formular wie folgt aus:

1. Wählen Sie als **Abwesenheitstyp** den Eintrag **„Krank"**.
2. Prüfen Sie, dass im Feld **Empfänger** **Philipp Dölker** eingetragen ist.
3. Aktivieren Sie **„Ganzer Tag"**, falls die Abwesenheit ganze Tage umfasst.
4. Tragen Sie **Startdatum** und **Enddatum** ein.
5. Geben Sie optional eine **Beschreibung** ein (z. B. „Krank").
6. Klicken Sie auf **„Abwesenheit beantragen"**.

![Krankmeldung ausfüllen](../assets/Krankheit/Bildschirmfoto%202026-04-29%20um%2022.35.50.png)

Die Krankmeldung wird anschließend an Philipp Dölker übermittelt.
